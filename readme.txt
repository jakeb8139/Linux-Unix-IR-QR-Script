This script is intended to collect volatile and select non-volatile data on a Linux/Unix operating system.

To execute the script, follow the instructions below:

	1.) Insert the USB into an available USB port on target machine.
	2.) Once the USB is mounted, open it via the left tool bar or desktop.
	3.) Right click inside the USB file system, and select "Open In Terminal".
	4.) Gain Root privileges by typing, "sudo su", pressing enter, and then entering the Root password.
	5.) In the command line with root privileges, type "./response" and press enter to execute the script.
	6.) Follow the prompts accordingly.
